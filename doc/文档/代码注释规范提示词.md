# 代码注释规范提示词

## 核心要求
1. **全代码注释**：为所有代码添加注释，包括公开接口、私有成员、内部实现、算法细节
2. **中英双文**：所有注释采用英文在前、中文在后的双语格式
3. **适量讲解**：为复杂算法和实现细节添加代码讲解（这是学习STL实现的项目）
4. **Doxygen风格**：使用标准Doxygen格式，便于生成文档
5. **无遗漏原则**：所有代码元素都必须有注释，包括简单的getter/setter、比较操作符、赋值操作符等

## 注释格式

### 1. 文件级注释
```cpp
/**
 * @file filename.hpp
 * @brief Brief description in English
 * @brief 中文简要描述
 * 
 * @author shensjt
 * @date YYYY-MM-DD
 */
```

### 2. 类注释
```cpp
/**
 * @brief Class description in English
 * @brief 类的中文描述
 * 
 * @tparam T Template parameter description
 * @tparam T 模板参数描述
 */
template<typename T>
class ClassName {
```

### 3. 函数注释
```cpp
/**
 * @brief Function description in English
 * @brief 函数的中文描述
 * 
 * @tparam Args Template parameters
 * @tparam Args 模板参数
 * @param args Parameter description
 * @param args 参数描述
 * @return Return value description
 * @return 返回值描述
 * 
 * @note Important notes
 * @note 重要说明
 */
template<typename... Args>
return_type function_name(Args&&... args) {
```

### 4. 成员变量注释
```cpp
pointer begin_;           ///< Pointer to the first element / 指向第一个元素的指针
pointer end_;             ///< Pointer to one past the last element / 指向最后一个元素之后位置的指针
pointer capacity_;        ///< Pointer to one past the allocated memory / 指向分配内存之后位置的指针
```

### 5. 操作符注释
```cpp
/**
 * @brief Equality comparison operator
 * @brief 相等比较操作符
 * 
 * @param other Another object to compare with
 * @param other 要比较的另一个对象
 * @return true if objects are equal, false otherwise
 * @return 如果对象相等则返回true，否则返回false
 */
bool operator==(const ClassName& other) const {
```

### 6. 内联注释（代码讲解）
在复杂算法处添加内联注释：
```cpp
// 检查是否需要扩容：如果当前已满，则重新分配内存
if (end_ == capacity_) {
    // 翻倍增长策略：新容量 = 当前容量 * 2（如果当前容量为0，则新容量为1）
    size_type new_capacity = capacity() == 0 ? 1 : capacity() * 2;
    reallocate(new_capacity);  // 调用重新分配函数
}
```

## 实施要点

### 1. 全面覆盖（无遗漏检查清单）
以下所有代码元素都必须有注释：
- [ ] **文件级注释**：每个源文件和头文件
- [ ] **命名空间注释**：每个命名空间
- [ ] **类/结构体注释**：每个类、结构体、联合体
- [ ] **枚举注释**：每个枚举类型和枚举值
- [ ] **类型别名注释**：每个using/typedef定义
- [ ] **成员变量注释**：所有公开、保护、私有成员变量
- [ ] **构造函数/析构函数注释**：所有构造函数、拷贝构造函数、移动构造函数、析构函数
- [ ] **成员函数注释**：所有公开、保护、私有成员函数
- [ ] **操作符重载注释**：所有操作符重载（==, !=, <, <=, >, >=, =, +=, -=等）
- [ ] **模板参数注释**：所有模板类和模板函数的模板参数
- [ ] **函数参数注释**：所有函数的参数
- [ ] **返回值注释**：所有函数的返回值
- [ ] **异常说明注释**：函数的异常安全性说明
- [ ] **复杂度说明注释**：关键算法的时间复杂度和空间复杂度
- [ ] **内联代码讲解**：复杂算法和关键实现细节的代码讲解

### 2. 双语格式
- 英文描述在前，中文翻译在后
- 使用相同的Doxygen命令（如`@brief`、`@param`、`@return`等）
- 确保中英文内容准确对应
- 技术术语保持一致性

### 3. 代码讲解
- 在复杂算法处添加内联注释
- 解释设计决策和实现原理
- 对于学习STL的项目，适当讲解标准库实现细节
- 说明算法的时间复杂度和空间复杂度

### 4. 简洁实用
- 注释要简洁明了，避免冗长
- 重点解释"为什么"和"如何工作"
- 为关键算法添加时间复杂度说明
- 避免重复显而易见的描述

## 示例

### 完整函数示例
```cpp
/**
 * @brief Constructs an element in-place at the end
 * @brief 在末尾原位构造元素
 * 
 * @tparam Args Types of arguments to forward to the constructor
 * @tparam Args 要转发给构造函数的参数类型
 * @param args Arguments to forward to the constructor
 * @param args 转发给构造函数的参数
 * @return Reference to the newly constructed element
 * @return 对新构造元素的引用
 * 
 * @note Time complexity: Amortized O(1)
 * @note 时间复杂度：摊还O(1)
 * 
 * @exception std::bad_alloc If memory allocation fails
 * @exception 如果内存分配失败则抛出std::bad_alloc
 */
template<typename... Args>
reference emplace_back(Args&&... args) {
    // 检查是否需要扩容：如果当前已满，则重新分配内存
    if (end_ == capacity_) {
        // 翻倍增长策略：确保摊还O(1)复杂度
        size_type new_capacity = capacity() == 0 ? 1 : capacity() * 2;
        reallocate(new_capacity);
    }
    
    // 在末尾位置构造元素，使用完美转发避免不必要的拷贝
    alloc_traits::construct(alloc_, end_, mystl::forward<Args>(args)...);
    ++end_;  // 更新end指针
    return back();
}
```

### 操作符示例
```cpp
/**
 * @brief Equality comparison operator
 * @brief 相等比较操作符
 * 
 * @param other Another vector to compare with
 * @param other 要比较的另一个vector
 * @return true if both vectors have the same size and elements are equal,
 * false otherwise
 * @return 如果两个vector大小相同且元素相等则返回true，否则返回false
 * 
 * @note Time complexity: O(min(n, m)) where n and m are the sizes of the vectors
 * @note 时间复杂度：O(min(n, m))，其中n和m是vector的大小
 */
bool operator==(const vector& other) const {
    if (size() != other.size()) return false;
    for (size_type i = 0; i < size(); ++i) {
        if ((*this)[i] != other[i]) return false;
    }
    return true;
}
```
